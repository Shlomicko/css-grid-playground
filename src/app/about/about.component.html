About this app

This is a set of simple CSS Grid examples I originally created for a talk at the 2018 Craft Conference called "CSS Grid: Get ready to fall in love" (although I've added a few things since then).

Note: The examples are hosted in an Angular app that uses Sass, but that's just to make some things easier to build or demonstrate. None of the CSS Grid concepts are specific to Angular or Sass.

You can find the latest version here: CSS Grid Playground on Github
